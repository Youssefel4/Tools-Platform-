# ✅ Google Indexing Checklist

## Phase 1: Initial Setup (Day 1)

### Google Search Console
- [ ] Create Google account (if needed)
- [ ] Go to https://search.google.com/search-console
- [ ] Add property: `https://platformtools.netlify.app`
- [ ] Choose verification method (HTML tag recommended)
- [ ] Copy verification code
- [ ] Add verification code to `public/index.html`
- [ ] Verify ownership
- [ ] Navigate to Sitemaps section
- [ ] Submit: `https://platformtools.netlify.app/sitemap.xml`
- [ ] Wait for sitemap processing (usually instant to few hours)

### Manual Indexing Requests
- [ ] Open URL Inspection tool in Search Console
- [ ] Request indexing for homepage: `/`
- [ ] Request indexing for `/calculator`
- [ ] Request indexing for `/unit-converter`
- [ ] Request indexing for `/password-generator`
- [ ] Request indexing for `/qr-generator`
